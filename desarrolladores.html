Sobre la aplicación:

Esta aplicacion consiste en un casino virtual donde los usuarios podrán jugar a la ruleta europea con apuestas ficticias.

Equipo de desarrollo:

-Ignacio Menares
Estudiantes de 2do año de la carrera de Ingenieria en informatica y Telecomunicaciones en la Universidad Diego Portales.

Tecnologías utilizadas:

Para este proyecto se hizo uso de dos herramientas fundamentales para la creación de páginas web, HTML para la estructura y CSS para el diseño visual.

Aclaraciones:

Esta aplicación fue desarrollada como parte de un proyecto académico, con fines educativos. No se maneja dinero real. El objetivo fue aplicar conocimientos de programación y trabajo colaborativo.
